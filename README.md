# shape-it-ob3


## Description

Code for shapeit-it with openbabel3


## INSTALL

- following example is basic way.

```
git clone https://github.com/iwatobipen/shape-it-ob3.git
cd shape-it-ob3
mkdir biuild
cd build
cmake -DCMAKE_INSTALL_PREFIX=<where you want to insatall> ..
make
make install
```
